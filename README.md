🚆 Railway Booking System
📌 Overview

The Railway Booking System is a Python-based console application that simulates an online railway reservation system. It allows users to search available trains, book tickets, generate unique PNR numbers, assign seats, and view booking details. The project demonstrates the use of Object-Oriented Programming (OOP) concepts, file handling, and data management in Python.

✨ Features
🔍 Search available trains
🎫 Book railway tickets
👤 Passenger information management
🪑 Automatic seat allocation
🆔 Unique PNR number generation
📄 Display booking details
❌ Ticket cancellation (if implemented)
💾 Data storage using file handling
🖥️ Simple and interactive console interface
